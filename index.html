A Jornada de Otho

A Aventura de um Desenvolvedor

Otho é um programador solitário em uma cidade digital. Sua missão é salvar sua cidade e derrotar os bugs que ameaçam sua existência.

Missão 1: Salvar o Código

Otho precisa corrigir o erro que bloqueou seu sistema. Ele usa o git bisect para rastrear o bug.

git bisect start
git bisect bad
git bisect good v1.0

Missão 2: Refatorar e Documentar

Agora, Otho deve refatorar seu código e adicionar comentários para facilitar a leitura.

# Função para calcular a soma dos números
def soma_numeros(lista):
    return sum(lista)

# Exemplo de uso
numeros = [1, 2, 3, 4, 5]
print(soma_numeros(numeros))  # Saída: 15

Missão 3: Criar um Pull Request

Otho envia suas alterações para o repositório GitHub com um pull request.

git checkout -b refatorar-codigo
git push origin refatorar-codigo
